# 🎓 DAIICT M.Sc. IT Resources

A curated collection of **DAIICT M.Sc. IT (Dhirubhai Ambani University)** notes, lecture slides, previous year question papers (PYQs), assignments, lab work, projects, books, and other study resources collected throughout my Master's journey.

Whether you're preparing for semester exams, looking for **DAIICT M.Sc. IT notes**, searching for **previous year papers (PYQs)**, or simply need additional study material, this repository aims to be a one-stop resource for current and future **M.Sc. IT students at Dhirubhai Ambani University**.

The repository will continue to grow with each semester as I upload new lecture notes, assignments, practicals, projects, and exam resources.
